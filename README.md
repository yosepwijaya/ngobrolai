# Ngobrol.ai

Production-grade AI dashboard framework.

- Backend: FastAPI, SQLAlchemy
- Frontend: React/Vite/Tailwind
- Google SSO, PostgreSQL
- Plug & play agent, doc, API, and report management

## Quickstart

```bash
# Backend
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload

# Frontend
cd frontend
npm install
npm run dev
```
